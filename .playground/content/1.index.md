# About

Welcome to the [SDS Documentation Site template](https://designsystem.sei.cmu.edu/templates/documentation-site)—a flexible, full-featured starting point for building high-quality documentation sites using modern web technologies.

This template is powered by **Nuxt Layer**, built on **Vue 3**, **Tailwind CSS** and the [SEI Design System](https://designsystem.sei.cmu.edu/). It's engineered for performance, accessibility, and extensibility, making it an ideal foundation for both internal and public-facing documentation. Look at the [Nuxt Content documentation](https://content.nuxtjs.org/) to learn more.

::callout{type="subtle" variant="orange"}
⚠️ This site uses **dummy content** about fake applications (FlowStack, Lume, ScribeAI, and VaultSync) to showcase the structure, layout, and component styling available in this template. Replace or extend this content with your own documentation.
::

## 🚀 Features at a glance

- ✅ **Structured layout** with clearly defined content regions  
- ✅ **Powerful navigation** with global top-level links, 3-level sidebar, and a table of contents
- ✅ [**Code block styling**](1.about/2.codeblocks.md) with syntax highlighting and quality indicators
- ✅ [**Callout styling**](1.about/3.callouts.md) for highlighting key information
- ✅ **Integrated search** for custom plugin integration   
- ✅ **Dark and light mode support**  
- ✅ **Fully responsive** and **WCAG-compliant**  
- ✅ **Powered by Nuxt Layer** with support for customization and scaling  

---

## 📦 Tech stack

- [**Vue 3**](https://vuejs.org/) – Fast, reactive UI components  
- [**Nuxt**](https://nuxt.com/) – Server-side rendering and static site generation  
- [**Tailwind CSS**](https://tailwindcss.com/) – Utility-first styling with responsive defaults  
- [**SEI Component Library**](https://designsystem.sei.cmu.edu/) – Accessible, design-consistent UI elements  
- [**Markdown**](https://www.markdownguide.org/) – Simple, readable content authoring

## 📚 Get started

Want to use this template for your own Nuxt documentation project?  


```bash
npx nuxi@latest init <project-name>
cd <project-name>
npm install @cmu-sei/sds-documentation-site
```
Check out the [**installation guide**](https://designsystem.sei.cmu.edu/help/documentation-site) to get up and running.
