# 1.1.0

*Release Date: 2025-09-03*

VaultSync 1.1.0 brings several high-impact enhancements focused on mobility, sharing flexibility, and administrative control. This update builds on our 1.0 foundation with new capabilities for mobile users, better external sharing, and improved visibility for teams managing secure collaboration.

---

## ✨ New Features

### Mobile Editing Support
- Users can now **edit files directly from mobile devices**
- Supports rich text, markdown, and common document formats (.docx, .txt, .md)
- Autosave and conflict resolution are fully supported on iOS and Android

### Bulk External Sharing
- Share multiple files and folders externally with **batch permissions**
- Set custom expiration dates and password protection per link
- View all active shares in the new **External Access Dashboard**

### Permission Analytics
- Visualize permission levels across users and teams at a glance
- Export access reports for auditing and compliance
- Easily identify overexposed files or unclaimed shares

---

## 🛠 Improvements

- 🧭 Redesigned sidebar for easier navigation and access to synced folders
- ⚡ Improved sync performance for directories with >10,000 files
- 📂 Better version history with visual diff comparison for text-based files
- 🔒 Enhanced admin controls for revoking access and resetting credentials

---

## 🧪 Known Issues

- Preview rendering may fail for large .csv files in mobile view (fix planned in 1.1.1)
- Some third-party integrations (e.g. Dropbox) still read-only
- Folder-level permission analytics limited to top-level directories

---

## 📌 Upgrade Notes

No migration required—existing users will receive this update automatically.  
New mobile features require the latest version of the VaultSync mobile app (v1.1 or later).
