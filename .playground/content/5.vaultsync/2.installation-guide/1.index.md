
# Installation Guide

Welcome to the VaultSync installation guide. This page walks you through system requirements, getting started steps, and workspace setup for secure file collaboration.

VaultSync supports both cloud-hosted and self-hosted deployment options, and works seamlessly across desktop, mobile, and web environments.

---

## 🖥️ System Requirements

### For End Users

- **Operating Systems**:  
  - Windows 10+  
  - macOS 12+  
  - Android 10+  
  - iOS 14+

- **Browser Support** (Web Access):  
  - Latest versions of Chrome, Firefox, Edge, or Safari

### For Admins (Self-Hosted Environments)

- **Node.js**: v18+  
- **Database**: PostgreSQL 13+ (recommended), or SQLite (for development)  
- **Storage**: Minimum 5 GB free disk space  
- **Memory**: 4 GB RAM minimum  
- **Docker**: Required for containerized deployments  
- **Network**: Outbound HTTPS access for connected services (Slack, email, etc.)

---

## 🚀 Installation Steps

### Option 1: Use VaultSync in the Cloud

> ✅ **Recommended for most users**

1. Visit [vaultsync.app/signup](https://vaultsync.app/signup)  
2. Create an account and choose a workspace name  
3. Invite team members via email or Slack  
4. Start uploading files and configuring permissions

No installation or configuration needed—you're ready to go!

---

### Option 2: Self-Hosted Installation (Advanced)

> ⚠️ Recommended for enterprise or air-gapped environments

 **Clone the repository**

::code-block
   ```bash
   git clone https://github.com/your-org/vaultsync
   cd vaultsync
