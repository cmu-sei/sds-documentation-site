# 1.0.0

*Release Date: 2025-08-06*

We're excited to announce the first stable release of **VaultSync**—a secure, cloud-based file collaboration and backup system designed to protect your data while keeping your teams in sync.

This release introduces the core functionality of VaultSync, delivering reliable syncing, encrypted storage, and user-friendly sharing tools out of the box.

---

## ✨ New Features

### End-to-End Encryption
- All files are encrypted in transit and at rest using AES-256
- Role-based access control ensures only authorized users can view or modify data

### Multi-Device Syncing
- Sync files across desktops, laptops, and mobile devices in real time
- Conflict resolution engine handles simultaneous edits with smart merging

### Secure File Sharing
- Share files with internal or external users via secure, time-limited links
- Set permissions for view, comment, or edit access

### Role-Based Permissions
- Define roles like admin, editor, viewer at folder and file levels
- Easily manage access across individuals and teams

### App Integrations
- Native support for Google Workspace and Microsoft 365
- Connect with Slack to get notifications on shared file activity

---

## 🛠 Improvements

- Optimized upload and sync performance for large files
- Improved file versioning and rollback controls
- Enhanced audit logs and activity tracking

---

## 🧪 Known Limitations

- Mobile app currently supports view-only access (editing support coming soon)
- External sharing is limited to single-use links (bulk permissions coming in 1.1.0)
- Real-time co-editing is supported only on desktop for now

---

## 📣 What's Next

We're actively working on the following for upcoming releases:

- Full mobile editing experience  
- Granular permission analytics  
- Integration with additional cloud providers (e.g. Dropbox, Box)
