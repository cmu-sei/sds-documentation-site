
# Automation Templates

FlowStack includes a growing library of pre-built automation templates that help teams get up and running quickly. These templates are fully customizable and designed to solve common operational challenges—no manual scripting required.

Use them as a starting point or inspiration to build your own reusable flows.

## Automating Emails & Notifications

Keep your team and stakeholders informed—automatically.

- Send templated emails when a task is created, updated, or overdue  
- Push Slack or Microsoft Teams messages for key workflow milestones  
- Set up digest-style daily or weekly summaries delivered to inboxes  
- Trigger alerts from external tools (e.g. GitHub PR merged, Jira ticket closed)  
- Customize message content using variables and conditional logic
