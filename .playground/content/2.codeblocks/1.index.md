## What are code blocks?

Including code is a critical part of technical documentation. The [SDS Documentation Site template](https://designsystem.sei.cmu.edu/templates/documentation-site) allows you to use both standard Markdown syntax and enhanced Vue-based components to create clear, structured, and visually helpful code blocks.

The template supports multiple ways to present code, from simple examples to fully styled instructional snippets. See our [examples page](/codeblocks/examples) to learn more!

## Why use code blocks?

Code blocks help:

- **Enhance clarity** by separating code from narrative text.
- **Facilitate copy-pasting** for developers referencing examples.
- **Support syntax highlighting** for multiple languages, improving readability and comprehension.
