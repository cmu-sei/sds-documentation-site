# Examples

The SDS Documentation Site Template supports multiple ways to present code, from simple examples to fully styled instructional snippets. 

## 🧱 Structure

### Simple

A basic code block that is written using triple backticks (```) in Markdown. Use this type of block for short, straightforward code examples that don’t require additional context or interaction.

```javascript
const hello = ref()
watch(hello, (value) => {
  alert(value)
})
```

### Interactive

Use this version when you want to enable developers to easily **copy-to-clipboard** or view the code **full-screen**.

::code-block
```javascript
const hello = ref()
watch(hello, (value) => {
  alert(value)
})
```
::

### Interactive with a title 

Use the `title` prop to add a brief description above your code block. This helps readers quickly understand the purpose or context of the snippet.

::code-block{title="Code example with a title"}
```javascript
const hello = ref()
watch(hello, (value) => {
  alert(value)
})
```
::


## 🎨 Visual quality indicators

You can visually label code blocks to represent best practices or problematic examples using the `quality` prop:

- `quality="good"` – Blue background to indicate correct usage and recommended practices  
- `quality="bad"` – Red or orange background for anti-patterns or errors

This visual distinction helps developers easily identify which snippets to follow—and which to avoid.

## 🦄 Additional features

### Expandable

An expandable code block that displays a partial snippet by default and includes an eye icon to reveal the complete code.

::code-block
```javascript
const hello = ref()
watch(hello, (value) => {
  alert(value)
})
// ...
```

#complete
```javascript
const hello = ref()
watch(hello, (value) => {
  alert(value)
})

alert('here is more code')
```
::



## 🧪 Supported languages

The `code-block` component currently supports syntax highlighting for:

- **Rust** – Known for safety and performance  
- **Python** – Ideal for simplicity and scripting  
- **Bash** – Commonly used for automation and shell scripting  
- **JavaScript**, **Java**, **C/C++**, **Perl** – And more

---

### Rust code block
::code-block{title="Rust example"}
```rust
// Function that returns multiple values
fn calculate_dimensions() -> (i32, i32) {
    (30, 50) // Returning a tuple
}

fn main() {
    let (width, height) = calculate_dimensions();
    println!("Width: {}, Height: {}", width, height);
}
```
::

### Python code block

::code-block{title="Python example"}
```python
# Function that returns multiple values
def calculate_dimensions():
    return (30, 50)  # Returning a tuple

# Call the function and unpack values
width, height = calculate_dimensions()
print("Width:", width)
print("Height:", height)
```
::

### Bash code block

::code-block{title="Bash example"}
```bash
#!/bin/bash
# Define a variable
greeting="Hello, World!"
# Print the variable
echo $greeting
```
::


### JavaScript code block

::code-block{title="JavaScript example"}
```javascript
// Function that returns multiple values using an array
function calculateDimensions() {
  return [30, 50];
}

const [width, height] = calculateDimensions();
console.log(`Width: ${width}, Height: ${height}`);
