# Features

::callout{type="subtle" variant="green"}
✅ **Note:** This documentation **is real** unlike the rest of the site that uses dummy content.
::

The SDS Documentation Site Template includes built-in features that help authors create clear, structured, and visually engaging documentation. These features support consistent formatting and improve how information is presented across all pages.

Use these content features to add emphasis, clarity, and context to your documentation.

## 💻 Code Blocks
[Code Blocks](/features/codeblocks) let you format and display snippets of code, commands, or configuration examples within your documentation. They support syntax highlighting, titles, and quality labels to help readers quickly understand and reuse examples.


## 💬 Callouts
[Callouts](/features/callouts) let you highlight important information, such as notes, tips, and cautions, directly within your content. They provide visual emphasis that helps readers quickly identify supporting details or critical guidance.
